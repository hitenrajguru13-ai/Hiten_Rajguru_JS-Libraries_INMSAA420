# 🏔️ Ridgeline Treks

A responsive trekking and adventure website created for the **JavaScript Libraries Assignment (INMSAA420)** at Seneca Polytechnic.

## 📖 Overview

Ridgeline Treks is a modern hiking website featuring interactive maps, animated charts, smooth scrolling effects, image galleries, and responsive design.

## ✨ Features

- Responsive design
- Hero section
- Interactive trekking gallery
- Leaflet interactive map
- Chart.js line chart
- GSAP animations
- AOS scroll animations
- Mobile-friendly navigation

## 🛠️ JavaScript Libraries Used

- Leaflet.js
- Chart.js
- GSAP
- AOS
- Glide.js

## 💻 Technologies

- HTML5
- CSS3
- JavaScript
- Node.js
- Lite Server

## ▶️ Run the Project

```bash
npm install
npm start
```

Open:

```
http://localhost:3000
```
